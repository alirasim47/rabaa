# 🎮 نظام الرابعة - إدارة الاشتراكات

## تشغيل النظام

### المتطلبات
- Node.js مثبت على الجهاز

### خطوات التشغيل

1. **فك الضغط** عن المجلد في أي مكان تريد

2. **افتح CMD أو Terminal** داخل مجلد المشروع

3. **ثبّت المكتبات** (مرة واحدة فقط):
   ```
   npm install
   ```

4. **شغّل السيرفر**:
   ```
   npm start
   ```

5. **افتح المتصفح** على:
   ```
   http://localhost:3000
   ```
